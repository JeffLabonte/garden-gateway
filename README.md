## Getting Started

This project currently runs only on Raspberry PI. 

It was tested on Debian Buster (Raspberry Pi OS)

### Create .env file 

1. Run `make copy_template`
2. Edit `.env` using your favorite editor
3. Update `DATABASE_URL`
